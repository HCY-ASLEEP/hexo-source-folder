用n升级nodejs<!--more-->

```bash
npm install -g n    # 全局安装n

n stable    # 升级到最新稳定版

n latest    # 升级到最新版

n v7.10.0    # 升级到定制版

n 7.10.0 (ENTER)    # 切换使用版本
```
